## Usage Tips

1. **Plan ahead** - Fill this out at the start of the week
2. **Mix it up** - Don't do the same type every day
3. **Batch create** - Write multiple posts in one sitting
4. **Schedule early** - Use the scheduler to queue them up
5. **Track results** - Note what performs best
6. **Iterate** - Adjust based on what your audience responds to

## Commands to Use

```bash
# Add a post
python add_post.py --platform nostr --content "..." --schedule "2026-03-12T10:00:00Z"

# View queue
python manage_queue.py list

# Run scheduler
python scheduler.py

# Daemon mode
python scheduler.py --daemon
```
